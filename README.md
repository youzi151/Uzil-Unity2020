# Uzil-Unity2020

## 簡介

自用工具集, 對應Unity版本2020.

以簡易, 方便開發使用為主, 不想做得太偏GUI或框架化.

盡量功能分拆, 減少"工具去干涉遊戲主循環"或者"主循環必須依循框架而寫"的影響.

各功能的目錄結構也分開, 避免工具與專案項目的檔案混淆.

距離近代Unity版本有一定落差, 引入與環境設置可能較容易出現問題.
